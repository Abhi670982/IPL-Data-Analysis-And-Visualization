# IPL Data Analysis using Python

## Overview

This project analyzes Indian Premier League (IPL) match and ball-by-ball datasets using Python. The objective is to perform Exploratory Data Analysis (EDA), generate insights, and visualize trends related to teams, players, toss decisions, match outcomes, and scoring patterns.

---

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook
* Git & GitHub

---

## Dataset

The project uses two IPL datasets:

### matches.csv

Contains match-level information such as:

* Season
* Teams
* Venue
* Toss winner
* Toss decision
* Match winner
* Player of the Match

### deliveries.csv

Contains ball-by-ball information such as:

* Match ID
* Innings
* Over
* Batsman
* Bowler
* Runs scored
* Wickets

## Analysis Performed

### Match Analysis

* Matches played in each season
* Runs scored in each season
* Runs per match across seasons
* Teams with the most wins
* Teams with the most toss wins

### Toss Analysis

* Toss decision distribution
* Toss decision variation across seasons
* Impact of toss on match results
* Chasing team win percentage

### Batting Analysis

* Leading run scorers
* Most fours and sixes
* Highest strike rates
* Most balls faced

### Bowling Analysis

* Leading wicket takers
* Team bowling statistics

### Team Analysis

* Highest team totals
* 200+ scores analysis
* Powerplay performance
* Death overs performance
